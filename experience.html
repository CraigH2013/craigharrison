Ball Aerospace

ballaerospace.com

  • Battlespace Visualization Laboratory

  • Created new visualization interfaces that provided informative metrics for operator augmentation. These interfaces worked both in playback and realtime to allowing an analyst to observe visualized data and annotate events in the dataset.
  • Collaborated with research psychologists to provide custom visualization dashboards for datasets ranging from F22 flight data, to Zephyr Systems bio data
  • Worked extensively with D3.js to map electroencephalogram (EEG) data to brain visualizations. This allowed analysts to gain quick insight into which parts of the brain were firing when doing specific tasks. My visualization work was used for our lab's cognitive brain research projects.

  • Space Operational Assessment for COA Development

  • Built an offline web application to manage space operation course of actions. Together our team was able to ingest OG50 reports for the purpose of quickly automating analyses in order to make decisons on critical space anomolies.
  • Created an administrative dashboard system to view all course of actions currently in the system. This helped analysts manage large amounts of data more easily.
  • Led a database redesign of our system. Our previous database models limited our ability to cross-refrence course of actions with their associated satellites. The new models helped create a "cleaner" code base and sped up development cycles.

  • Cyber Security Research

  • Developed embedded secure system on Xilinx Zedboard, 702 and ODROID SoC boards using SierraWare and Xen hypervisors. The purpose of our research was to explore the idea of creating secure, trusted, environments inside the hypervisor.
  • Modified open-source hardware emulator QEMU and OpenBIOS firmware to properly support Motorola 8270 processor. This allowed us to run legacy flight managment system (FMS) software in our trusted environments.
  • Adapted Flight Management System (FMS) software binary to run in a QEMU-emulated PowerPC environment. Created supervised communication systems between the FMS and our embedded secure systems.

  • Cypber Physical Systems

  • Created network algorithms to parse cyber attack graphs. These algorithms found optimal paths through the graphs while scoring each path based on effectivness through the system.
  • Built a D3.js visualization API for Java. This allowed our developers to use web visualizations in a JavaFX application. I built an interproccess communication system from the ground up to allow interaction with the visualiations and their respective network code.
  • Designed custom visualizations for multi-dimensional networks. The visualization allowed graphs to have children and show detail as you clicked into each node of the graph. This allowed for simplified views of a complex network. The user could click on nodes and 'open ' them up to see more detail in the graph.
