Experience

Ball Aerospace

ballaerospace.com

  • Battlespace Visualization Laboratory

  • Created new display interfaces that provided informative metrics for operator augmentation. These interfaces worked both in playback and real-time to allow an analyst to observe visualized data and annotate events in the dataset.
  • Collaborated with research psychologists to provide custom visualization dashboards for datasets ranging from F22 flight data to Zephyr Systems bio data
  • Worked extensively with D3.js to map electroencephalogram (EEG) data to brain visualizations. These displays allowed analysts to gain quick insight into which parts of the brain were firing when doing specific tasks.

  • Space Operational Assessment for COA Development

  • Built an offline web application to manage space operation course of actions. Together our team was able to ingest OG50 reports for the purpose of quickly automating analyses to make decisions on critical space anomalies.
  • Created an administrative dashboard system to view all course of actions currently in the system. Analysts were able to manage large amounts of data more efficiently because of this tool.
  • Led a database redesign of our system. Our previous database models limited our ability to cross-reference course of actions with their associated satellites. The new models helped create a "cleaner" code base and sped up development cycles.

  • Cyber Security Research

  • Developed embedded secure system on Xilinx Zedboard, 702 and ODROID SoC boards using SierraWare and Xen hypervisors. The purpose of our research was to explore the idea of creating safe, trusted, environments inside the hypervisor.
  • Modified open-source hardware emulator QEMU and OpenBIOS firmware to adequately support Motorola 8270 processor. With these improvements, we could run legacy Flight Management System (FMS) software in our trusted environments.
  • Adapted Flight Management System (FMS) software binary to run in a QEMU-emulated PowerPC environment. Created supervised communication systems between the FMS and our embedded secure systems.

  • Cyber-Physical Systems

  • Created network algorithms to parse cyber attack graphs. These algorithms found optimal routes through the networks while scoring each path based on effectiveness through the system.
  • Built a D3.js visualization API for Java. Our developers could use this API to display web visualizations in a JavaFX application. I built an interprocess communication system from the ground up to allow interaction with the visualizations and their respective network code.
  • Designed custom visualizations for multi-dimensional systems. The display allowed graphs to have children and show detail as you clicked into each node of the graph.
